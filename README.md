# housing-market-analysis-poland

## Opis projektu
Projekt zawiera analizę Polskiego rynku mieszkaniowego z naciskiem na analizę cen mieszkań, oraz kredytów hipotecznych.

## Źródła
GUS
AMRON-SARFiN
NBP
Otodom

## Metody
Czyszczenie i przygotowanie danych
Analiza danych
Wieloraka regresja liniowa
Wizualizacja danych

## Narzędzia
Microsoft Excel (Power Pivot, wykresy, VLOOKUP, REGLINP)
